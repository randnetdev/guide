Hello. I'm going to show you section 1 of how to install DoraOS.
DoraOS was made a anonymous developer who wishes to not be named, but was made specifcally for Dora's project.
First off, you're going to want to ask Dora for the firmware download link, and you should recieve a ZSD file.
From there, you're going to want to install this and run the installer, and hit Install. You'll also want to install Win32DiskImager, which can be found here and lastly, you'll want to install Putty, which can be found here
From there, you're set to begin. Right click on the zsd file and go to 7Zip-ZS>Extract files... and specify a location where you want your firmware to be extracted.
Then, open Win32DiskImager, and select the 32GB SD Card you are flashing to (IT WILL NOT WORK WITH ANY SD CARDS SMALLER), then select the DoraOS_BG3LN79ZEFHGXDGTVTA88CNUDD4RCVKK_Firmv1.img file, and hit write. This will write it in a raw format to your SD card.
